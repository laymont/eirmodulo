Ayaguna

Que es Ayaguna

Ayaguna Contro de Contenedores, es un aplicación Web desarrollada con los lenguajes HTML 5, PHP 5, MySQL, Javascripts y JQuery. Como su nombre lo indica esta orientada al control de contenedores maritimos. Su principal ventaja se basa en que es una herramienta en linea, lo cual le permite a su empresa poder tener a la mano la información necesaria con tan solo contar con una dispositivo con conexión a internet y un navegador. Tanto la empresa como sus clientes tendra en tiempo real la información que necesitan en cualquier momento, contando con esta las 24 hrs del dia, los 365 dias del año.

Ayaguna es una aplicación completamente amigable y con posibilidad de ajustarse a las necesidades de cada empresa.

Comenzando

Antes de comenzar a dar uso a la aplicación, el usuario Administrador debe completar ciertas tareas. Se deben registrar: Los Usuarios, las Lineas, los Buques, los Viajes. La aplicación cuenta con la opción de importar lista de descarga de buque, para de este modo facilitar la introducción de los datos y minimizar el margen de error en el registro de los datos. Una vez completadas las tareas antes indicadas ya pueden comenzar a dar uso a la aplicación.

Tracking

Este modulo se encuetra en la pantalla de inicio de la aplicacion, el campo a traves del cual se accede a este, tiene la etiqueta "Busqueda"; ingrese el numero del contenedor que esta buscando. Luego de hacer click en el boton enviar, obtendra la informacion del equipo que esta buscando; se mostraran los N cantidad de movimientos (registros) que posea el contenedor en el sistema. El campo IN/OUT de la tabla donde se muestra la información, indica si el contenedor esta dentro (In) o fuera (Out) del inventario.

Editar datos del contenedor

Para editar los datos del contenedor debera acceder a traves del tracking, una vez que obtenga la respuesta, notara que el numero de contenedor aparece subrayado; haciendo click sobre el numero del contenedor podra editar los datos del contenedor.

Ingresos - Vacios

Modulo para registrar el movimiento de ingreso de contenedores vacios; el usuario debera completar los datos necesarios para registrar el ingreso del contenedor vacio. Los datos requeridos son:

  • EIR
  • Linea
  • Buque
  • Viaje
  • Equipo
  • Tipo
  • Condición
  • Fecha Muelle (Despacho Pto.)
  • Fecha de Recepción (Alm.).
  • Consignatario
  • Ubicación
  • Observaciones

La version 2.0 de Ayaguna cuenta con comportamientos del lado del cliente para prevenir el ingreso de contenedores que ya esten registrados en el inventario. Asi como tambien valida el numero del contenedor segun la norma ISO-6346, el campo Equipo del formulario de ingreso es el encargado de informar sobre la validacion del numero del equipo.

Actualmente la aplicación se encuentra en la planificación y desarrollo del modulo para la emisión de EIR.

Ingreso - Precargado

Gracias a el modúlo de Precarga, la aplicación puede importar las lista de descargas emitidas por las lineas; desde esta lista precargada se puede ingresar de manera individual los datos de los contenedores, a los cuales se les debe anexar los datos faltantes para completar el resgistro del ingreso.

Para quitar los estilos en línea, asegúrese de que el panel Estilos CSS está configurado como Actual. Seleccione la imagen y, en la sección Propiedades del panel Estilos CSS, haga clic con el botón derecho del ratón y elimine las propiedades de visualización y de fondo. (Por supuesto que siempre podrá ir directamente al código y eliminar allí los estilos en línea de la imagen o el marcador de posición.)
